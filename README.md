---

## MVP Scope

- Universe: Nifty 50 companies
- Data source: Manually ingested financials (annual)
- Flags: 5 core red flags (rule-based, explainable)
- Output: Company-wise red flag summary

---

## Initial Red Flags (MVP)

1. Operating Cash Flow < Net Profit (persistent)
2. Rising Debt with Falling Revenue
3. Interest Coverage Deterioration
4. Promoter Holding Decline
5. Negative Free Cash Flow Trend

---

## Philosophy

- Deterministic logic (no black-box models)
- Fully explainable flags
- Built for long-term investors
- Risk visibility > return prediction
